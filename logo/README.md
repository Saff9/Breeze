# Breeze Logo

The Breeze mark: three wind lines with a curl — the universal glyph for a
breeze — in a teal rounded square.

## Files

| File | Use |
|------|-----|
| `breeze.svg` | The icon. Use for app icons, favicons, extension icons. |
| `breeze-wordmark.svg` | Icon + "Breeze" wordmark. Use in READMEs, headers. |
| `favicon.svg` | Small 64×64 variant for browser tabs. |

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| Teal 400 | `#2dd4bf` | Gradient start |
| Teal 600 | `#0d9488` | Gradient end |
| White | `#ffffff` | Wind lines |
| Slate 900 | `#0f172a` | Wordmark text |

## Guidelines

- Don't stretch or skew the mark.
- Keep clear space around it equal to the curl height.
- On dark backgrounds, the gradient still reads well; no inverted variant needed.
- Minimum size: 16×16 for the icon alone.
